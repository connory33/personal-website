Autonomous Robot

This was a project in circuits, Arduino, and bit-level C code to build an autonomous robot to compete in a cube-clearing competition against another robot.





Planned a strategy of attack for the competition, constructed multiple H-bridge circuits on breadboards for wheel control with pulse width modulation. Integrated a color sensor for pathing decisions, designed and laser-cut a protective housing and snowplow component for better clearing capabilities. Programmed the robot to operate fully autonomously using Arduino and C with bits, registers, etc.
