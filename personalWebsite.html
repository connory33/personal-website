Personal Website

I've only recently begun web development, but it's something that has gained my interest very quickly. I intend for this website to be complementary with my learning. I'm working on learning more webdev tools like JavaScript, React, and back-end languages like Java. I hope to use these to improve this site as I improve with them.


Ideally what results is an interesting tale of my web-development education experience seen through the evolution of this website. Check back periodically to see what's changed!
