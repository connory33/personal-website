Personal Website

I built this website in July 2019 to teach myself basic web development while also creating a platform to showcase my abilities and projects. It's a work in progress that will continue to improve as time goes on. Check out my GitHub here to see the full commit history.

I wanted to teach myself web development the right way, from the ground up. So, I built this site using nothing but vanilla HTML, CSS, and JavaScript. As my skill level progresses and I become comfortable with the basics, I plan to add on frameworks like Bootstrap and React to both teach myself the skills as well as improve my site. The site is independently hosted with CPanel, not using any helper services like WordPress.
